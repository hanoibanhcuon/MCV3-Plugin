# HR Skeleton — Nhân sự / Quản lý lao động

## Skeleton BIZ-POLICY-HR

```markdown
## BR-HR-001: Chính sách chấm công
**Quy tắc:** [Giờ làm việc, tolerance check-in/out?]
**Áp dụng cho:** Tất cả nhân viên
**Xử lý đi trễ:** [Trừ lương? Cảnh báo?]

## BR-HR-002: Phê duyệt nghỉ phép
**Quy tắc:** Nghỉ phép cần báo trước [?] ngày và phê duyệt của [ai?]
**Ngoại lệ:** Nghỉ khẩn cấp → báo trong [?] giờ

## BR-HR-003: Tính lương
**Chu kỳ:** [Ngày ?/tháng]
**Công thức lương:** [Cố định / Cố định + KPI / Hoa hồng]
**Các khoản khấu trừ:** [BHXH, BHYT, BHTN, thuế TNCN]
**Phụ cấp:** [Ăn trưa, xăng xe, điện thoại...]
```

## Skeleton PROCESS-HR

### PROC-HR-001: Tuyển dụng

```
Tạo yêu cầu tuyển dụng → Phê duyệt quản lý
  → Đăng tin → Nhận hồ sơ
  → Sàng lọc → Phỏng vấn → Chọn ứng viên
  → Offer letter → Onboarding
```

### PROC-HR-002: Tính lương hàng tháng

```
Thu thập dữ liệu chấm công (ngày cuối tháng)
  → Tính giờ OT, nghỉ phép
  → Apply bảng lương cơ bản + phụ cấp
  → Tính BHXH, thuế TNCN
  → Duyệt bảng lương (Kế toán → Giám đốc)
  → Chuyển khoản
  → Gửi phiếu lương
```

## Hỏi user về chính sách đặc thù

```
1. "Công ty có chính sách OT (làm thêm giờ) như thế nào?"
2. "Có shift/ca làm việc không? Bao nhiêu ca?"
3. "Commission/hoa hồng tính theo công thức nào?"
4. "Có KPI gắn với lương không?"
```
